# Source Key

Each card cites its sources in two layers.
The **Source basis** entries name the one to three writings the card’s guidance was
synthesized from, with the role each played.
The **Corroboration** entry lists additional, independently checked locations in the
published Buffett and Berkshire record — and in two widely available books — where the
same point is stated, applied, or qualified.
Corroboration citations are abbreviated locations, not quotations, and they are not a
substitute for reading the original writing.

## Abbreviated Citation Forms

| Key | Source |
| --- | --- |
| `BH<year>` | Berkshire Hathaway annual shareholder letter for that year (e.g., `BH2002`) |
| `BPL<year>` | Buffett Partnership letter covering that year, from the 1957–1970 partnership-letter compilation |
| `OM1996` | Warren Buffett, *An Owner’s Manual* (1996) |
| `SI1984` | Warren Buffett, “The Superinvestors of Graham-and-Doddsville” (1984) |
| `WB50-2015` | Warren Buffett, “Berkshire — Past, Present and Future,” 50th-anniversary essay (2015) |
| `CM50-2015` | Charlie Munger, “Vice Chairman’s Thoughts — Past and Future,” companion 50th-anniversary essay (2015) |
| `FL2025` | Warren Buffett, farewell letter to shareholders (November 2025) |
| `GC<year>` | Warren Buffett, annual charitable-giving comments for that year (June) |
| `SC<year>` | Warren Buffett, November shareholder comments for that year |
| `ACL2022` | Warren Buffett, Activision reporting-correction letter (2022) |
| `ABM2010` | Warren Buffett, Class A and Class B relative-price memo (2010) |
| `IL2001` | Warren Buffett, interim shareholder letter (November 2001) |
| `EWB §<section>` | *The Essays of Warren Buffett: Lessons for Corporate America* (Buffett, arranged by Lawrence A. Cunningham, 1998), cited by essay section (e.g., `EWB §II.A`, Mr. Market) |
| `LOW ch.<n>` | Roger Lowenstein, *Buffett: The Making of an American Capitalist*, cited by chapter |

## How to Read Corroboration Entries

- A parenthetical after a key narrows the citation to the specific aspect it supports,
  for example `LOW ch.18 (Coca-Cola franchise case)`.
- `EWB` sections arrange Buffett’s own shareholder-letter text from 1979–1996. When an
  `EWB` citation accompanies a Source-basis entry from the same period, it shows that
  the point was also selected as thematically central in that arrangement; it is not an
  independent author.
- `LOW` is a secondary biography.
  Its citations corroborate what Buffett and Berkshire did in a historical case, not
  that Buffett stated the card’s guidance.
- All other keys cite Buffett’s or Munger’s own published writings.

Corroboration entries were reconciled against the project’s full source collection,
including sources published after the material the cards were first synthesized from.
They provide representative, not exhaustive, coverage; check the original writing and
the company evidence under analysis before relying on any card.
